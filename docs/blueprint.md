# **App Name**: StockPilot

## Core Features:

- QR Code Scanning: Scan QR codes using the device's camera to quickly identify products.
- Stock In/Out: Record stock entries and exits by scanning products. Track quantities.
- Stock Management: Monitor product stock levels, including the quantity of each item.
- Location Tracking: Identify where the product is such as depot or shelf
- PWA Support: Enable the app as a Progressive Web App for mobile-like experience.
- Database Storage: Stock data stored to track product locations and history.

## Style Guidelines:

- Primary color: Deep sky blue (#43bfff) to give the application a tech feel.
- Background color: Very light blue (#E2F3FF).
- Accent color: Turquoise (#71f2a7) for calls to action and highlights to draw user attention to the most important tasks and functionalities in the app
- Body and headline font: 'Inter' sans-serif for a clean and modern user interface.
- Use minimalist, clear icons to represent actions and categories. Use icons related to each stock location.
- Optimize layout for mobile screens with a focus on ease of use. Design it to use one-handed on the go while in a depot environment
- Incorporate smooth transitions between screens.
